# Introduction à Next.js 15

Next.js 15 apporte de nombreuses améliorations et nouvelles fonctionnalités pour le développement d'applications React modernes.

## Les nouvelles fonctionnalités

### App Router amélioré
L'App Router continue d'évoluer avec de meilleures performances et une API plus intuitive.

```typescript
// Exemple de page avec App Router
export default function Page() {
  return (
    <div>
      <h1>Ma page Next.js</h1>
    </div>
  )
}
```

### Server Components par défaut
Les Server Components sont maintenant utilisés par défaut, offrant de meilleures performances.

## Tableau des fonctionnalités

| Fonctionnalité | Next.js 14 | Next.js 15 |
|---------------|------------|------------|
| App Router | ✅ | ✅ Amélioré |
| Server Components | ✅ | ✅ Par défaut |
| Turbopack | 🚧 Beta | ✅ Stable |

## Conclusion

Next.js 15 représente une évolution majeure pour le développement d'applications React performantes.
