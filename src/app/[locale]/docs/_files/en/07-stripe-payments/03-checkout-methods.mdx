## Unified Checkout Page

All checkout methods are rendered through the main checkout page:

```typescript
// From src/components/features/checkout-stripe/checkout-page.tsx
export default async function CheckoutPage({
  priceId,
  couponId,
  seats = 1,
  guest = false,
  enableInstallments = false,
}: {
  priceId: string
  couponId: string
  seats: number
  guest: boolean
  enableInstallments?: boolean
})
```

// From src/components/features/checkout-stripe/checkout-page.tsx
const checkoutConfig = getCheckoutConfig({

### Guest Checkout

All Custom System checkout methods support guest checkout with automatic user account creation:

```typescript
// Enable guest checkout
<CheckoutPage
  priceId="price_123"
  couponId=""
  seats={1}
  guest={true} // Allow checkout without account creation
/>
```

**Guest Checkout Workflow:**
1. **Guest Payment:** User completes payment without creating account
2. **Webhook Processing:** `handleGuestCheckoutSessionCompleted` function processes the event
3. **User Creation:** System automatically creates user account from checkout data
4. **Subscription Linking:** Links the new user to their paid subscription
5. **Account Access:** User can access their account and manage subscription

**Technical Implementation:**

<Tabs defaultValue="checkout-detection">
  <Tab value="checkout-detection" label="🔍 Detection">
    ```typescript
    // From src/lib/stripe/stripe-events.ts
    function detectGuestCheckoutMode(metadata: Record<string, string>): boolean {
      const referenceId = metadata.referenceId
      const guestCheckoutFlag = metadata.guest_checkout === 'true'

      const isGuestByReferenceId = referenceId === 'guest'
      const isGuestByFlag = guestCheckoutFlag
      const isGuestCheckout = isGuestByReferenceId || isGuestByFlag

      return isGuestCheckout
    }
    ```
  </Tab>

  <Tab value="user-creation" label="👤 User Creation">
    ```typescript
    // From src/lib/stripe/stripe-events.ts
    async function createGuestUser(customerData: {
      customerEmail: string
      customerName?: string
      stripeCustomerId?: string
    }) {
      // 1. Ensure Stripe customer exists
      const finalStripeCustomerId = await ensureStripeCustomer(customerData)

      // 2. Create or update database user
      const finalUser = await ensureDatabaseUser(
        customerData.customerEmail,
        customerData.customerName,
        finalStripeCustomerId
      )

      return finalUser
    }
    ```
  </Tab>

  <Tab value="subscription-linking" label="🔗 Subscription Linking">
    ```typescript
    // From src/lib/stripe/stripe-events.ts
    async function updateSubscriptionWithUser(
      subscriptionId: string,
      user: { id: string; email: string; stripeCustomerId: string }
    ) {
      // Use billing context to determine correct referenceId
      const { referenceId } = await getBillingContext(user.id)

      const updatedSubscription = await updateSubscriptionForWebhookService({
        subscriptionId,
        referenceId,
        stripeCustomerId: user.stripeCustomerId,
      })
    }
    ```
  </Tab>
</Tabs>

**Benefits:**
- **Zero Friction:** No account creation required during checkout
- **Higher Conversion:** Reduces checkout abandonment
- **Automatic Account:** User gets account access post-purchase
- **Seamless Experience:** User can manage subscription immediately
- **Data Integrity:** Proper linking between user, customer, and subscription

## Checkout URL Parameters (Custom System)

The Custom System checkout page at `/checkout/[priceId]` supports various URL parameters:

<Callout type="warning">
  **Better Auth users:** URL parameters are not available with Better Auth's external checkout system.
</Callout>

```typescript
// From src/app/[locale]/(public)/checkout/[priceId]/page.tsx
interface CheckoutParams {
  priceId: string        // Stripe Price ID (required)
  couponCode?: string    // Coupon/discount code
  seats?: number         // Number of seats for team subscriptions
  guest?: 'true'         // Enable guest checkout
  split?: 'true'         // Enable installment payments
}
```

**Example URLs:**
```bash
# Basic checkout
/checkout/price_1234567890

# Guest checkout with coupon
/checkout/price_1234567890?guest=true&couponCode=SAVE20

# Team subscription with 5 seats
/checkout/price_1234567890?seats=5

# Installment checkout for guest
/checkout/price_1234567890?guest=true&split=true
```
